# 学习方法 Demo

## 科目概述

这是一个用于演示 Review 三种模式的中性资料包，主题是有效学习方法。学习者需要区分常见学习策略的适用场景，并能判断一种学习安排是否有助于长期记忆和迁移。

## 考试目标

- 能区分主动回忆和重复阅读的效果差异。
- 能解释间隔复习为什么优于临时突击。
- 能判断错题复盘是否抓住了错误根因。
- 能把知识卡片、做题和复盘组合成一个学习闭环。

## 出题风格

- 题型偏好：判断、选择、简答均可。
- 难度分布：S-U 和 M-U 为主，少量 M-A。
- 出题特点：重辨析、重场景判断。
- 代码语言：none
