# 01 - TUI 长内容与滚动修复

状态：已实现，待真实 pi TUI 手动复验。

## 目标

修复复习过程中长题干、代码块、章节材料、卡片内容在 TUI 中被省略、截断或滚动回顶的问题。用户可见内容必须完整可读，尤其是 `review_answer` 的题目正文。

## 实施范围

- `review_answer` 使用可滚动题目面板展示完整 `question_text`，再进入答案输入。
- 卡片、章节、考点材料共用稳定的窗口化渲染逻辑，保留当前滚动位置。
- 删除用户可见题干中的“已省略 N 行”展示策略；工具返回给 agent 的文本可以短，但 TUI 面板不能丢内容。
- 长行按宽度拆分展示，避免左侧内容消失。

## 验收

- 含 50 行代码块的简答题可滚动阅读后作答。
- 卡片和章节材料滚动后不会自动回到顶部。
- 找不到章节材料时能显示缺失提示，不崩溃。
- `npm run check` 与 `npm test` 通过。
