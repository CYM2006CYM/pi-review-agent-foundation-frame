# C++标准

C++ 标准决定语言语法和标准库特性的可用范围。课程画面列出的标准演进包括 C++98、C++03、C++11、C++14、C++17。

## 时间线

| 标准 | 课程画面标注 |
|---|---|
| C++98 | major |
| C++03 | fixed bug |
| C++11 | major |
| C++14 | minor |
| C++17 | major |

## 复习意义

> [!important] 重点
> 同一段 C++ 代码在不同标准下可能表现不同，甚至无法编译。做题或写代码时要注意题目指定的 C++ 标准。
