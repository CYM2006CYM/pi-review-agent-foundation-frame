# C++语言

[[C++语言]]是本课程讲解[[面向对象程序设计]]的主要实现语言。它既继承 C 语言的过程式和结构化基础，也支持类、对象、继承、多态、模板、STL 等机制。

## 为什么适合本课程

- 支持[[面向对象程序设计]]：类、对象、继承、多态。
- 支持[[泛型编程]]：模板、STL。
- 支持多种[[编程范型]]，便于比较不同程序设计思想。
- 标准持续演进，需要关注[[C++标准]]版本。

## 易错点

> [!warning] 易错
> C++ 支持面向对象，但不等于 C++ 只能面向对象。它是一门多范型语言。
