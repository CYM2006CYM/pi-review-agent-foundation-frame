# RTTI

RTTI 是 Runtime Type Information，即运行时类型识别。

它让程序在运行期间获得对象真实类型信息，通常依赖含有[[虚函数]]的多态类型和[[虚函数表]]关联的类型信息结构。

常见用法：

- [[typeid]]：获得表达式的类型信息；
- [[dynamic_cast]]：安全地做运行时向下类型转换。

RTTI 有运行时查找成本。能直接用[[多态]]和虚函数分派表达的行为，不应滥用类型判断。
