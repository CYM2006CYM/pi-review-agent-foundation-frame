# is-a关系

[[is-a关系]] 表示“某个子类是一种父类”。它是判断是否应该使用[[公有继承]]的核心语义标准。

例如：

```text
手动挡轿车是一种轿车
自动挡轿车是一种轿车
```

如果这句话在自然语言中说不通，就不应使用公有继承，即使语法上可以编译。
