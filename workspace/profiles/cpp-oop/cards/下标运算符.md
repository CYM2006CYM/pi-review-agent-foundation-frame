# 下标运算符

[[下标运算符]] `[]` 常用于通过下标访问对象内部元素。

```cpp
int operator[](int index) const;
int& operator[](int index);
```

非 const 版本常返回引用，以支持：

```cpp
a[i] = value;
```

`operator[]` 必须是成员函数。
