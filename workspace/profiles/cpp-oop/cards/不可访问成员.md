# 不可访问成员

[[不可访问成员]] 指派生类对象中存在，但派生类代码不能直接使用名字访问的基类成员。典型情况是基类的 `private` 成员。

它们仍然必须保留在派生类对象中，因为继承来的基类成员函数可能需要访问这些私有数据。

```cpp
class Base {
private:
    int num;
public:
    void set(int n) { num = n; }
};

class Derived : public Base {
public:
    void f() {
        set(10);  // 合法：调用基类 public 成员函数
        // num = 10; // 非法：不能直接访问 Base::num
    }
};
```
