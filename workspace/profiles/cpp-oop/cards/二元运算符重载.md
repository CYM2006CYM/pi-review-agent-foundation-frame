# 二元运算符重载

[[二元运算符重载]] 为有两个操作数的运算符定义自定义类型行为。

自由函数形式：

```cpp
T operator+(const T& lhs, const T& rhs);
```

成员函数形式：

```cpp
T operator+(const T& rhs) const;
```

自由函数中左右操作数都是显式参数；成员函数中左操作数是 `*this`。
