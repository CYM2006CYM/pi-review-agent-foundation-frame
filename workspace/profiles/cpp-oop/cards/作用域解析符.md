# 作用域解析符

[[作用域解析符]] `::` 用于指明一个名字属于哪个作用域、类或名字空间。

类外实现成员函数时常用：

```cpp
void Student::study(int hours) {
}
```

`Student::study` 表示 `Student` 类中的 `study` 成员函数。

在名字空间中也常用：

```cpp
std::cout << "hello";
My::myFunc();
::globalFunc();
```

`::globalFunc()` 中左侧为空，表示从[[全局名字空间]]查找。
