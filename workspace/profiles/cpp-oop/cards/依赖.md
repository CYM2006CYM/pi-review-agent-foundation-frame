# 依赖

[[依赖]] 是类之间较弱的水平逻辑关系，通常表现为一个类在函数参数、返回值或函数实现中使用另一个类。

```cpp
class B {
public:
    void f(A* pA);
    A* g();
};
```

如果 `A` 不是 `B` 的数据成员，而只是某次函数调用或函数实现中用到，通常归为依赖。

依赖的耦合强度低于关联。能用依赖表达的关系，通常不应随意升级成关联。
