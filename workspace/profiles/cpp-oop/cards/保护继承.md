# 保护继承

[[保护继承]] 使用 `protected` 作为继承方式。

```cpp
class Derived : protected Base {
};
```

在保护继承中，基类的 `public` 和 `protected` 成员在派生类中都变成 `protected`；基类的 `private` 成员仍不可直接访问。

保护继承比公有继承更强调“内部复用”，不把基类的公开接口完整暴露给派生类对象的外部使用者。

相关：[[派生类访问控制]]、[[protected]]、[[白盒复用]]
