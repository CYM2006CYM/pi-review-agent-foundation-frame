# 公有继承

[[公有继承]] 使用 `public` 作为继承方式。

```cpp
class Derived : public Base {
};
```

在公有继承中，基类的 `public` 成员在派生类中仍为 `public`，基类的 `protected` 成员仍为 `protected`，基类的 `private` 成员在派生类中不可直接访问。

设计上，公有继承通常用于表达“派生类是一种基类”的关系，因此更适合称为父类/子类关系。

相关：[[派生类访问控制]]、[[父类]]、[[子类]]
