# 关联

[[关联]] 是类之间的一种水平逻辑关系，通常表现为一个类把另一个类作为数据成员保存。

```cpp
class B {
private:
    A* pA;
};
```

因为这种关系保存在对象结构中，所以通常比只在函数调用中出现的[[依赖]]更强。

关联可以进一步分为[[一般关联]]和[[聚集关联]]。如果关系强调整体-部分，还要继续区分[[聚合]]和[[组合]]。
