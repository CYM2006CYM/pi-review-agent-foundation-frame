# 关联与继承

[[关联与继承]] 是把水平关系和垂直关系组合起来的设计方式：一个类通过数据成员关联某个父类接口，而具体对象由该父类的不同派生类提供。

典型形式：

```cpp
class A {
private:
    Parent* p;
};
```

其中 `Parent` 可以有多个派生类。`A` 只依赖 `Parent` 的稳定接口，派生类变化不要求修改 `A`。

适用场景：

- 当前类长期持有或使用某类对象；
- 被持有对象有多个种类，并且未来可能增加；
- 希望把新增种类的影响限制在被关联对象的继承体系中。
