# 函数对象

[[函数对象]] 是可以像函数一样调用的对象，通常通过重载 [[函数调用运算符]] 实现。

```cpp
struct Add {
    int operator()(int a, int b) const {
        return a + b;
    }
};
```

使用：

```cpp
Add add;
int x = add(1, 2);
```
