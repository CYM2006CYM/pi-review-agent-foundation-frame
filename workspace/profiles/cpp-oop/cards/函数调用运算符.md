# 函数调用运算符

[[函数调用运算符]] `()` 让对象可以像函数一样被调用。

```cpp
class A {
public:
    int operator()(int a, int b) const;
};
```

使用：

```cpp
A a;
a(10, 20);
```

重载了 `operator()` 的对象称为 [[函数对象]]。`operator()` 必须是成员函数。
