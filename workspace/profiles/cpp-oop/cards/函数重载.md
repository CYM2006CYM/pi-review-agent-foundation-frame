# 函数重载

[[函数重载]]指多个函数使用相同函数名，但参数列表等可区分。

```cpp
void f(int);
void f(double);
```

返回类型和默认参数不能作为重载区分标志。

在继承体系中要额外小心：派生类一旦定义了与基类同名的函数，可能触发[[同名隐藏]]，使基类中所有同名重载函数在派生类中被隐藏。
