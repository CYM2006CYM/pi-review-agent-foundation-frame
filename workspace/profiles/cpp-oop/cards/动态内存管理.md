# 动态内存管理

[[动态内存管理]] 是程序在运行期间按需要创建和释放对象的机制。

在 C++ 中，基础形式是：

```cpp
T* p = new T(参数列表);
delete p;
```

它通常发生在 [[自由存储区]] 或 [[堆区]]，能弥补静态存储在对象数量、大小和生命周期上的不足。

风险包括 [[内存泄漏]]、[[重复释放]]、[[悬空指针]] 和 `new/delete` 不匹配。
