# 单例模式

[[单例模式]] 是限制某个类最多只创建一个实例的设计方式。

一个基础写法是：

```cpp
class A {
public:
    static A& instance() {
        static A object;
        return object;
    }

private:
    A();
};
```

关键点：

- 构造函数私有化。
- 通过类方法提供创建或访问入口。
- 使用静态局部对象保存唯一实例。

在类设计中，单例通常通过[[私有构造函数]]、禁止拷贝/赋值、公开静态 `getInstance` 方法实现。
