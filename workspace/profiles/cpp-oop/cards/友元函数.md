# 友元函数

[[友元函数]] 不是类的成员函数，但被类授予访问其私有成员的权限。

```cpp
class A {
    friend A operator+(const A& lhs, const A& rhs);

private:
    int x;
};
```

友元函数常用于需要访问私有成员的 [[非成员运算符重载]]。

友元关系是单向的、没有传递性。友元声明不受 `public`、`private`、`protected` 位置影响，因为友元本身不是类成员。
