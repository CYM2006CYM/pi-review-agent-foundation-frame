# 友元类

[[友元类]] 是被某个类整体授予访问其私有成员权限的类。

```cpp
class Wallet {
    friend class Parent;

private:
    int money;
};
```

`friend class Parent;` 表示 `Parent` 的所有成员函数都可以访问 `Wallet` 的私有成员。

特点：

- 授权范围比单个[[友元函数]]更大。
- 友元关系是单向的。
- 友元关系没有传递性。
- 使用时应谨慎，避免过度破坏[[封装]]。
