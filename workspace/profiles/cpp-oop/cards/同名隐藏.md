# 同名隐藏

[[同名隐藏]] 指派生类中定义了某个与基类成员函数同名的函数后，基类中所有同名函数在派生类作用域中被隐藏。课程中也把这种现象称为 `overwrite(hide)`。

```cpp
class Base {
public:
    void f();
    void f(int);
};

class Derived : public Base {
public:
    void f(double);

    void test() {
        // f();    // 可能找不到 Base::f()
        // f(10);  // 可能找不到 Base::f(int)
        f(3.14);
    }
};
```

如果确实需要让基类同名函数参与查找，可以使用[[using声明]]：

```cpp
class Derived : public Base {
public:
    using Base::f;
    void f(double);
};
```

> [!warning] 易错点
> 同名隐藏只看函数名，不要求参数列表相同。派生类定义一个 `f(double)`，也可能隐藏基类中的 `f()` 和 `f(int)`。

相关：[[函数重载]]、[[using声明]]、[[redefine]]
