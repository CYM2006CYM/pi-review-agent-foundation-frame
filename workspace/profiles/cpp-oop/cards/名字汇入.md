# 名字汇入

[[名字汇入]] 是把某个名字空间中的名字引入当前作用域，减少完整限定名书写。

两种常见形式：

```cpp
using namespace std; // 汇入整个名字空间中的名字
using std::cout;     // 只汇入指定名字
```

汇入的是“名字”，这个名字可能代表变量、函数、类或子名字空间。
