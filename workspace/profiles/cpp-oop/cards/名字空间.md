# 名字空间

[[名字空间]] 是 C++ 中组织名字、避免[[命名冲突]]的机制。它给变量名、函数名、类名等增加一层归属范围。

```cpp
namespace My {
    void f();
    class T {};
}

My::f();
```

同一个简单名字可以出现在不同名字空间中，只要完整限定名不同即可，例如 `Zhang::f` 和 `Li::f`。
