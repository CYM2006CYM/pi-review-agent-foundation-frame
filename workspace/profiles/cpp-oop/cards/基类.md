# 基类

[[基类]] 是被继承的已有类，也常称为父类的技术基础。派生类通过[[继承列表]]指出自己从哪些基类继承。

```cpp
class Derived : public Base {
};
```

这里 `Base` 是基类，`Derived` 是[[派生类]]。

> [!note] 术语区分
> “基类/派生类”是 C++ 继承语法中的通用说法；“父类/子类”通常更适合在[[公有继承]]语义下使用。

相关：[[派生类]]、[[公有继承]]、[[多重继承]]
