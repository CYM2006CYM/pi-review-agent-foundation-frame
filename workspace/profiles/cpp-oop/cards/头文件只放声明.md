# 头文件只放声明

[[头文件只放声明]]是 C++ 工程中的基本组织原则：公共头文件负责告诉其他编译单元“有什么名字可以用”，真正的普通变量和普通函数定义放在 `.cpp` 实现文件中。

典型写法：

```cpp
// counter.h
extern int count;
int next_count();
```

```cpp
// counter.cpp
int count = 0;
int next_count() {
    return ++count;
}
```
