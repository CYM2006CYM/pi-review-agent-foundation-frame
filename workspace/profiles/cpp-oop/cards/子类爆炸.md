# 子类爆炸

[[子类爆炸]] 指多个独立变化方向被硬塞进同一棵继承树后，派生类数量按组合数快速增长。

例如 `f` 有 2 种实现，`g` 有 3 种实现，`h` 有 4 种实现，若用一个继承树穷举组合，就需要 `2 x 3 x 4 = 24` 个子类。

解决思路是用[[关联]]或[[依赖]]拆分变化方向，每个协作对象负责一个方向。
