# 定位分配

[[定位分配]]是指在程序员指定的一块已有内存上构造对象。C++ 中常用 placement new 表达：

```cpp
char data[sizeof(A)];
A* p = new (data) A(1);
```

这里不会重新申请内存，而是在 `data` 所表示的存储空间上调用 `A` 的构造函数。

使用定位分配时要注意：

- 它强调“构造位置可控”。
- 底层内存可以来自栈区缓冲区或堆区缓冲区。
- placement new 构造出的对象通常需要显式调用析构函数。

```cpp
p->~A();
```
