# 实例变量

[[实例变量]] 是属于某个具体对象的数据成员。

```cpp
class Student {
public:
    int age;
    int score;
};
```

每创建一个对象，这些非静态数据成员就随对象各有一份。

```cpp
Student a;
Student b;

a.age = 18;
b.age = 20;
```

`a.age` 和 `b.age` 是两份不同的数据。
