# 实例常量

[[实例常量]] 是对象拥有的常量数据成员。

```cpp
class Student {
public:
    const int id;
};
```

它仍然属于某个具体对象，只是对象创建完成后不应再被修改。

实例常量和 [[类变量]] 的区别是：实例常量是每个对象各有一份；类变量是整个类共享一份。
