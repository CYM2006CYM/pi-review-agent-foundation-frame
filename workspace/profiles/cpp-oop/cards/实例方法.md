# 实例方法

[[实例方法]] 是通过具体对象调用的普通成员函数。

```cpp
class Student {
public:
    void study();
};

Student s;
s.study();
```

实例方法有隐含的 [[this指针]]，因此可以直接访问当前对象的 [[实例变量]]。
