# 实现文件

[[实现文件]]通常使用 `.cpp` 扩展名，用来存放函数定义、类成员函数实现、`main` 函数和具体算法逻辑。

实现文件通常是 C++ 工程中的基本[[编译单元]]。它会包含必要的[[头文件]]，经过[[预编译]]和[[编译]]生成[[目标文件]]，再通过[[链接]]形成最终程序。

关联：[[头文件]]、[[编译单元]]、[[2.9 实现文件]]
