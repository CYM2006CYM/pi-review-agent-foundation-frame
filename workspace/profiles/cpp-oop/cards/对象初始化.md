# 对象初始化

[[对象初始化]] 是对象创建时为其数据成员建立初始状态的过程。

构造函数体内的赋值语句并不等同于真正初始化；当执行到构造函数体时，对象已经创建完成。

```cpp
class A {
public:
    A() {
        x = 0; // 赋值
    }

private:
    int x;
};
```

对于 `const` 成员、引用成员等，必须使用真正的初始化方式，而不能依赖函数体赋值。
