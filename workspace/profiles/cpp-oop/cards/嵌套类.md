# 嵌套类

[[嵌套类]] 是定义在另一个类内部的类。

```cpp
class Outer {
public:
    class Inner {
    };
};

Outer::Inner obj;
```

嵌套类的名字受外部类访问控制影响；嵌套类内部仍按普通类的访问控制规则工作。

常见用途是把只服务于外部类的辅助类型隐藏起来，减少外部可见名字。
