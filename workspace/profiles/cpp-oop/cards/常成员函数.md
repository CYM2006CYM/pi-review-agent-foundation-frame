# 常成员函数

[[常成员函数]] 是承诺不修改当前对象状态的成员函数，通常在参数列表后写 `const`。

```cpp
class Student {
public:
    int score() const;
};
```

常成员函数中的 `const` 修饰隐含的 [[this指针]]，使其类似：

```cpp
const Student* const this
```

因此常成员函数不能修改当前对象的普通数据成员。[[常对象]] 只能调用常成员函数。
