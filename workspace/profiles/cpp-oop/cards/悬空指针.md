# 悬空指针

[[悬空指针]] 是仍保存某个地址，但该地址对应对象已经销毁或内存已经释放的指针。

```cpp
A* p = new A;
delete p;
// 此时 p 仍保存旧地址，但旧地址已无效
```

`delete p;` 释放的是指针指向的对象和内存，不会修改指针变量 `p` 自身的值。因此释放之后继续执行 `p->f()` 之类的访问，即使语法上可能通过，也已经属于错误访问。

常见处理习惯：

```cpp
delete p;
p = nullptr;
```
