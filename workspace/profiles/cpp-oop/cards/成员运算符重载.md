# 成员运算符重载

[[成员运算符重载]] 是把运算符函数写成类的成员函数。

```cpp
class A {
public:
    A operator+(const A& rhs) const;
};
```

表达式：

```cpp
a + b
```

会被理解成：

```cpp
a.operator+(b);
```

左操作数由隐含的 [[this指针]] 表示。
