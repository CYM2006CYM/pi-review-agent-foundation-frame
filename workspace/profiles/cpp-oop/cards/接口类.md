# 接口类

[[接口类]] 是主要由公有[[纯虚函数]]组成、用于描述公共行为契约的[[抽象类]]，通常不保存实例变量。

```cpp
class IComputer {
public:
    virtual ~IComputer() = default;
    virtual void runApp() = 0;
};
```

接口类的意义是规定“能做什么”，具体“怎么做”由派生类实现。

在 C++ 中没有单独的 `interface` 关键字，但可以用抽象类表达接口。
