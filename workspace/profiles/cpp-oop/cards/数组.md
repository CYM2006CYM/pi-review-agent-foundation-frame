# 数组

[[数组]]是一组相同类型元素按顺序连续组织的数据结构，元素在内存中通常[[连续存储]]。

在 C++ 类型体系中，数组属于由已有类型导出的类型。

```cpp
int a[5];
```

`a` 是包含 5 个 `int` 元素的数组，下标从 `0` 开始。
