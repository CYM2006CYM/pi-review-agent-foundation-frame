# 智能指针

[[智能指针]] 是用对象封装裸指针，并通过对象生命周期自动管理资源释放的类。

核心思想是 [[RAII]]：

```cpp
std::shared_ptr<T> p(new T);
```

智能指针通常重载 `operator->` 和 `operator*`，使其像普通指针一样使用。

相关：[[共享指针]]、[[RAII]]、[[动态内存管理]]、[[11.6 智能指针和共享指针]]
