# 构造函数

[[构造函数]] 是创建对象时自动调用的特殊成员函数，用于初始化对象。

```cpp
class Student {
public:
    Student();
};
```

构造函数名与类名相同，没有返回值类型，可以重载，也可以被设置为 `public`、`protected` 或 `private`。
