# 构造函数初始化列表

[[构造函数初始化列表]] 位于构造函数参数列表之后、函数体之前，用于指定成员如何构造。

```cpp
class A {
public:
    A() : number_(0), b2_(2) {}

private:
    int number_;
    B b1_;
    B b2_;
};
```

成员实际构造顺序由类中声明顺序决定，不由初始化列表书写顺序决定。
