# 派生类拷贝构造函数

[[派生类拷贝构造函数]] 负责用一个已有派生类对象构造另一个派生类对象。自定义时要同时处理[[基类子对象]]和派生类新增数据。

推荐写法是在初始化列表中显式调用基类拷贝构造函数：

```cpp
class B : public A {
public:
    B(const B& other) : A(other), number(other.number) {
    }
private:
    int number;
};
```

`A(other)` 会把 `other` 中的基类部分交给 `A` 的拷贝构造函数处理。
