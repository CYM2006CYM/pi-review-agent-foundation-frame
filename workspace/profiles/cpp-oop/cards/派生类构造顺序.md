# 派生类构造顺序

[[派生类构造顺序]] 是：先构造基类部分，再构造派生类自己的成员，最后执行派生类构造函数体。

对于多个基类，基类构造顺序由继承列表中的声明顺序决定。

```cpp
class Derived : public Base {
public:
    Derived(int n1, int n2) : Base(n1), value(n2) {
    }
private:
    int value;
};
```

如果基类没有无参构造函数，派生类构造函数必须在初始化列表中明确说明如何构造基类。

相关：[[派生类析构顺序]]、[[构造函数]]、[[初始化列表]]
