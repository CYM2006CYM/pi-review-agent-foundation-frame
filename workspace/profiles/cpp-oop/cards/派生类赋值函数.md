# 派生类赋值函数

[[派生类赋值函数]] 负责把一个已有派生类对象赋值给另一个已有派生类对象。自定义时要先处理自赋值，再处理基类部分和派生类新增部分。

```cpp
class B : public A {
public:
    B& operator=(const B& other) {
        if (this != &other) {
            A::operator=(other);
            number = other.number;
        }
        return *this;
    }
private:
    int number;
};
```

`A::operator=(other)` 调用基类赋值函数，处理对象中的基类部分。
