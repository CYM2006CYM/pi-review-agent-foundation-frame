# 深拷贝

[[深拷贝]] 不只是复制指针值，还会复制指针指向的资源，使新旧对象各自拥有独立资源。

```cpp
B::B(const B& other) {
    pa_ = new A(*other.pa_);
}
```

深拷贝常用于类直接管理堆区资源的场景。
