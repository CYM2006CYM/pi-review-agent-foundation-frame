# 私有继承

[[私有继承]] 使用 `private` 作为继承方式。如果 `class` 的继承列表中只写基类名而省略继承方式，默认就是私有继承。

```cpp
class Derived : private Base {
};
```

在私有继承中，基类的 `public` 和 `protected` 成员在派生类中都变成 `private`；基类的 `private` 成员仍不可直接访问。

私有继承通常更接近“用基类实现派生类”的内部复用，不应轻易理解为父类/子类关系。

相关：[[派生类访问控制]]、[[继承列表]]、[[白盒复用]]
