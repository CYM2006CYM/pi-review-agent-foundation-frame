# 箭头运算符

[[箭头运算符]] `->` 用于通过指针访问对象成员。

```cpp
Object* p = &object;
p->f();
```

它等价于：

```cpp
(*p).f();
```

在运算符重载中，`operator->` 必须返回指针，或返回另一个也能继续使用 `->` 的对象。
