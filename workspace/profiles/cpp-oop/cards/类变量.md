# 类变量

[[类变量]] 是属于类本身、被所有对象共享的数据。

C++ 中通常用 [[静态数据成员]] 表示类变量：

```cpp
class Example {
public:
    static int number;
};

int Example::number = 0;
```

类变量不属于单个对象，因此不计入单个对象的普通存储大小。
