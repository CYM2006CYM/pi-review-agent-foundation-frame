# 类方法

[[类方法]] 是属于类本身的方法。

C++ 中通常用 [[静态成员函数]] 表示类方法：

```cpp
class Example {
public:
    static int nextNumber();
};

Example::nextNumber();
```

类方法不依赖某个具体对象，因此没有隐含的 [[this指针]]。

在第 14 章中，类方法还可用于[[创建方法]]和[[单例模式]]。
