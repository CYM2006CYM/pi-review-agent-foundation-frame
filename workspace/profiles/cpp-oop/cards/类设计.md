# 类设计

[[类设计]] 是把问题域中的对象抽象成 C++ 类的过程。

基本步骤：

1. 找出对象需要描述的数据。
2. 找出对象需要提供的行为。
3. 判断数据属于对象还是属于类。
4. 判断行为依赖对象还是只依赖类。
5. 判断哪些成员应保持不变，哪些成员允许修改。
6. 根据语义选择 `const`、`static`、访问控制、内联或外联实现。

设计类时不要只问“能不能写”，还要问“这样写是否符合对象的语义”。
