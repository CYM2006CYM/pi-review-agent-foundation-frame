# 纯虚函数

[[纯虚函数]] 是声明末尾带 `= 0` 的虚函数。

```cpp
class Shape {
public:
    virtual double area() = 0;
};
```

只要类中至少有一个纯虚函数，这个类就是[[抽象类]]。

派生类若没有实现所有继承来的纯虚函数，仍然是抽象类。
