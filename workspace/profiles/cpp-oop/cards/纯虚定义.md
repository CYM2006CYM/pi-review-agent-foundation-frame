# 纯虚定义

[[纯虚定义]] 是给[[纯虚函数]]在类外提供函数体。

```cpp
class Shape {
public:
    virtual double area() = 0;
};

double Shape::area() {
    return 0;
}
```

有纯虚定义不代表该函数不再纯虚；声明仍是 `= 0`，所在类仍然是[[抽象类]]。

相关：[[18.7 具体类和抽象类]]、[[纯虚函数]]
