# 继承

[[继承]] 是 C++ 中实现[[白盒复用]]的主要机制。一个类可以从已有类派生出新类，使新类拥有基类中除特殊函数以外的大量成员，并在此基础上增加新的数据和行为。

继承关注两个层面：

- 语法层面：`class Derived : public Base { ... };`
- 设计层面：是否存在合理的类型层次，尤其在公有继承中是否能表达“子类是一种父类”。

> [!warning] 易错点
> 继承不是简单复制代码。派生类确实拥有基类成员，但基类的构造函数、析构函数、拷贝构造函数和赋值函数不会作为普通成员自动继承。
