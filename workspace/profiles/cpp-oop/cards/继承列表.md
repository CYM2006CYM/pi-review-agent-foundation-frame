# 继承列表

[[继承列表]] 写在派生类名之后，用来说明派生类从哪些基类继承，以及采用什么继承方式。

```cpp
class Child : public Parent1, private Parent2 {
public:
    Child(int n);
};
```

每一项通常由“继承方式 + 基类名”组成。继承方式可以是 `public`、`protected` 或 `private`；如果在 `class` 派生中省略继承方式，默认是[[私有继承]]。

相关：[[公有继承]]、[[保护继承]]、[[私有继承]]、[[多重继承]]
