# 自我赋值

[[自我赋值]] 指一个对象赋值给自身。

```cpp
a = a;
```

更隐蔽的情况是引用别名：

```cpp
B b1;
B& b2 = b1;
b1 = b2;
```

虽然变量名不同，但 `b1` 和 `b2` 表示同一个对象。

自定义赋值函数中通常用地址比较判断自我赋值：

```cpp
if (this != &rhs) {
    // 释放旧资源并复制新资源
}
return *this;
```

如果不判断自我赋值，资源类对象可能先释放自己的资源，再从同一块已经释放的资源读取，导致未定义行为。
