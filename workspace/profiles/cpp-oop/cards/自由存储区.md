# 自由存储区

[[自由存储区]] 是 C++ 中通过 `new` 和 `delete` 管理动态对象的存储区域，课程中也常与 [[堆区]] 一起理解。

```cpp
A* p = new A;
delete p;
```

自由存储区对象的生命周期由程序员控制。

相关：[[堆区]]、[[动态内存管理]]、[[new运算符]]、[[delete运算符]]、[[11.3 单对象的动态分配及释放]]
