# 虚拟拷贝构造函数

[[虚拟拷贝构造函数]] 是用普通虚函数 `clone()` 模拟“按动态类型复制对象”的技术。

构造函数和拷贝构造函数不能是虚函数，因此常写：

```cpp
class Base {
public:
    virtual ~Base() = default;
    virtual Base* clone() const {
        return new Base(*this);
    }
};
```

派生类重写 `clone()`，内部调用自己的拷贝构造函数。
