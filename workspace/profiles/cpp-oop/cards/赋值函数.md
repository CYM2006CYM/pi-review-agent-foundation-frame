# 赋值函数

[[赋值函数]] 用于处理对象之间的赋值。

```cpp
class A {
public:
    A& operator=(const A& other);
};
```

它解决的是“已有对象如何接收另一个已有对象状态”的问题。

赋值函数的常见完整要求包括：

- 参数通常写成 `const A& other`，避免修改右侧对象并避免额外拷贝。
- 返回类型通常写成 `A&`，用于支持 [[链式赋值]]。
- 函数体末尾通常返回 `*this`。
- 对资源类对象，要避免 [[浅赋值]]，并处理 [[自我赋值]]。
