# 转换函数

[[转换函数]] 是定义在类内部、用于把本类对象转换成其他类型的特殊成员函数。

```cpp
class Fraction {
public:
    operator int() const {
        return num / den;
    }

private:
    int num;
    int den;
};
```

特点：

- 格式是 `operator 目标类型() const`。
- 不写普通返回类型。
- 通常需要返回目标类型的值。
- 可用[[explicit关键字]]禁止隐式调用。
