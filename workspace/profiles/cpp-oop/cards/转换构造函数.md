# 转换构造函数

[[转换构造函数]] 是能够用一个实参把其他类型对象转换成本类对象的构造函数。

```cpp
class A {
public:
    A(int value);
};
```

在某些上下文中，编译器可能隐式调用它：

```cpp
void f(const A& a);
f(100); // 可能先构造临时 A(100)
```

如果不希望发生这种隐式转换，应使用 [[explicit关键字]]。

它写在目标类型中：如果要把 `A` 转成 `B`，可以在 `B` 中写 `B(const A&)`。
