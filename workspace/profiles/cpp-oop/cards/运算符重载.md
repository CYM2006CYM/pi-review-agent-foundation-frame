# 运算符重载

[[运算符重载]] 是为已有运算符在自定义类型上的使用定义函数。

```cpp
T operator+(const T& lhs, const T& rhs);
```

它不能创造新运算符，也不能改变运算符的操作数个数、优先级和结合性。至少一个操作数必须是自定义类型。
