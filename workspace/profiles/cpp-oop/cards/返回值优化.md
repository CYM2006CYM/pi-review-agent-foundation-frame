# 返回值优化

[[返回值优化]] 是编译器在函数按值返回对象时，省略不必要临时对象和拷贝的一类优化。

```cpp
A makeA() {
    return A(1, 2);
}
```

在运算符重载中，普通 `operator+` 常按值返回新对象；这不应被误改成返回局部对象引用。
