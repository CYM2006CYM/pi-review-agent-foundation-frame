# 递归组合结构

[[递归组合结构]] 指一个抽象父类既有普通叶子子类，又有容器子类；容器子类本身也是父类的一种，同时还能包含多个父类对象。

典型结构：

```cpp
class Component {};
class Leaf : public Component {};

class Composite : public Component {
private:
    std::vector<Component*> children;
};
```

这种结构适合表达树形整体-部分关系，例如公司和部门、文件夹和文件、图形对象和子图形对象。
