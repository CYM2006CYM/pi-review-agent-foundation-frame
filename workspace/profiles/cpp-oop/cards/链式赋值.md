# 链式赋值

[[链式赋值]] 是多个赋值表达式连续使用的形式。

```cpp
a = b = c;
```

它按右结合方式理解为：

```cpp
a = (b = c);
```

因此，`b = c` 的结果还要继续作为 `a = ...` 的右侧表达式。

对象赋值函数通常返回当前对象引用：

```cpp
T& operator=(const T& rhs);
```

这样赋值表达式才可以继续参与外层赋值。
