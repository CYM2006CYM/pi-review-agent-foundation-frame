# 隐式类型转换

[[隐式类型转换]] 是编译器在没有显式转换语法时，为了匹配函数参数、赋值目标或表达式运算而自动插入的类型转换。

```cpp
void f(float x);

int n = 10;
f(n); // int 可隐式转换为 float
```

自定义类型中，单参数构造函数和转换函数都可能参与隐式类型转换。如果不希望这种自动转换，应考虑使用[[explicit关键字]]。
