# 静态局部变量

[[静态局部变量]]是在函数或局部作用域中用 `static` 定义的变量。

```cpp
void f() {
    static int count = 0;
    ++count;
}
```

它的名字只在局部作用域内可见，但对象通常存放在[[全局数据区]]，[[生命周期]]持续到程序结束，并且只在第一次执行到定义处时初始化。
