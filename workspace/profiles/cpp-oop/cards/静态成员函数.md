# 静态成员函数

[[静态成员函数]] 是用 `static` 声明的成员函数，属于类本身。

```cpp
class Example {
public:
    static int nextNumber();
};
```

静态成员函数没有隐含的 [[this指针]]，所以不能直接访问 [[实例变量]]，也不能直接调用 [[实例方法]]。

它可以直接访问 [[静态数据成员]]，也可以调用其他静态成员函数。

推荐使用类名调用：

```cpp
Example::nextNumber();
```
