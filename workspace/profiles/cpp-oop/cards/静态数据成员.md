# 静态数据成员

[[静态数据成员]] 属于类本身，而不是某一个具体对象。

```cpp
class A {
private:
    static int count_;
};
```

它常用来表示 [[类变量]]，所有对象共享同一份。

静态数据成员不计入单个普通对象的大小，通常需要在类外单独定义：

```cpp
int A::count_ = 0;
```
