# 非成员运算符重载

[[非成员运算符重载]] 是把运算符函数写成类外函数。

```cpp
A operator+(const A& lhs, const A& rhs);
```

表达式：

```cpp
a + b
```

会被理解成：

```cpp
operator+(a, b);
```

普通二元运算符常用非成员形式，使左右操作数更对称。如果需要访问私有成员，可以配合 [[友元函数]]。
