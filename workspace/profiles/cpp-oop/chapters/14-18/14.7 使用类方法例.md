# 14.7 使用类方法例

本节通过两个例子说明[[类方法]]的用途：[[创建方法]]和[[单例模式]]。

## 例 1：用创建方法代替语义不清的重载构造函数

假设 `Balloon` 有多个构造函数：

```cpp
class Balloon {
public:
    Balloon();
    Balloon(int color);
    Balloon(Gas& gas);
};
```

语法上没有问题，但使用者看到多个构造函数时，往往需要查文档才能理解每个参数的语义。构造函数名必须和类名相同，不能通过函数名表达“普通气球”“彩色气球”“充气气球”等区别。

可以改成私有构造函数 + 公有类方法：

```cpp
class Balloon {
public:
    static Balloon createNormalBalloon()
    {
        return Balloon();
    }

    static Balloon createColorBalloon(int color)
    {
        return Balloon(color);
    }

    static Balloon createGasBalloon(Gas& gas)
    {
        return Balloon(gas);
    }

private:
    Balloon();
    explicit Balloon(int color);
    explicit Balloon(Gas& gas);
};
```

这样做的好处：

- 函数名自解释，语义清楚。
- 使用者不必只靠参数类型猜构造意图。
- 创建逻辑集中在类内部，后续扩展更方便。

> [!important] 高频考点
> 当多个重载构造函数语义不清时，可以用命名清楚的静态创建方法改善接口可读性。

## 例 2：单例模式

如果希望一个类最多只能创建一个对象，可以使用[[单例模式]]。

基本思路：

1. 把构造函数设为 `private`。
2. 禁止拷贝构造和赋值。
3. 提供一个 `public static` 方法返回唯一实例。

### 静态局部对象方式

```cpp
class Ball {
public:
    static Ball& getInstance()
    {
        static Ball ball;
        return ball;
    }

private:
    Ball() = default;
    Ball(const Ball&) = delete;
    Ball& operator=(const Ball&) = delete;
};
```

第一次调用 `getInstance` 时创建 `ball`，之后每次调用都返回同一个对象。

### 静态指针方式

```cpp
class Ball {
public:
    static Ball* getInstance()
    {
        if (pBall == nullptr) {
            pBall = new Ball;
        }
        return pBall;
    }

    static void releaseInstance()
    {
        delete pBall;
        pBall = nullptr;
    }

private:
    Ball() = default;
    Ball(const Ball&) = delete;
    Ball& operator=(const Ball&) = delete;

    static Ball* pBall;
};
```

静态指针方式更灵活，但要记得释放堆区对象，并在释放后把指针置空。否则再次获取实例时可能返回已经释放的地址。

> [!warning] 易错点
> 单例不是“程序员约定只创建一次”，而是类本身通过访问控制和类方法限制外部只能拿到同一个实例。

## 两种单例实现的比较

| 实现 | 优点 | 注意点 |
|---|---|---|
| 静态局部对象 | 写法简单，不需要手动释放 | 大对象会占静态存储，复杂控制不灵活 |
| 静态指针 | 灵活，适合延迟创建和手动释放 | 必须处理释放、置空、多线程同步 |

课程中更推荐理解静态指针方式，因为它更能体现“创建、保存、释放唯一实例”的完整过程。

## 本节小结

- 类方法可以用于表达属于类整体的行为。
- 创建方法能让对象创建语义更清楚。
- 私有构造函数可以限制类外直接创建对象。
- 单例模式通过静态方法返回唯一实例。
- 静态指针单例需要显式释放，并把指针置空。
