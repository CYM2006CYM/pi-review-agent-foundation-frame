# 3.9 表达式

## 本节核心

本节补充 C++ 中几个容易被忽略的[[表达式]]问题。

- 表达式通常会产生一个值，即使这个值在语句中被丢弃。
- `for` 循环的初始化、条件、步进部分本质上都是表达式位置。
- [[逗号表达式]]不是简单分隔符，它规定了从左到右的求值顺序，并把最后一个表达式的值作为结果。
- [[赋值表达式]]采用右结合，因此可以连续赋值。
- 能放在赋值号左边的表达式通常是[[左值表达式]]；只能求值使用的表达式通常是[[右值表达式]]。
- [[Lambda表达式]]是现代 C++ 以及许多现代语言中的重要表达式形式。

> [!important] 核心认识
> 表达式不只是“算一下”。它还涉及返回值、求值顺序、结合方向、能否作为赋值目标等语义。

## 表达式有值

表达式通常会产生结果。

例如：

```cpp
2 + 3;
```

这是一条合法的表达式语句。`2 + 3` 的结果是 `5`，只是这个结果没有被使用，直接被丢弃了。

再如：

```cpp
int x = 0;
x = 100;
```

`x = 100` 不只是“把 100 放进 x”，它本身也是一个[[赋值表达式]]，并且也有表达式结果。

> [!tip] 初学者理解
> 写成语句后没有接收结果，不代表表达式没有值，只是值被忽略。

## for 循环中的三个表达式

典型 `for` 循环：

```cpp
for (int i = 0; i < 10; ++i) {
    // 循环体
}
```

括号中有三个部分：

| 部分 | 含义 | 执行时机 |
|---|---|---|
| 初始化表达式 | 设置循环初始状态 | 进入循环前执行一次 |
| 条件表达式 | 判断是否继续循环 | 每轮循环体执行前判断 |
| 步进表达式 | 更新循环状态 | 每轮循环体执行后执行 |

执行顺序是：

1. 先执行初始化表达式。
2. 判断条件表达式。
3. 条件为真则执行循环体。
4. 执行步进表达式。
5. 回到条件表达式继续判断。

## 空表达式与死循环

`for` 的三个表达式都可以为空：

```cpp
for (;;) {
    // 无限循环
}
```

这里初始化表达式、条件表达式、步进表达式都为空。条件表达式为空时，`for` 会把它当作恒真条件，因此形成死循环。

它和下面写法类似：

```cpp
while (true) {
    // 无限循环
}
```

## 逗号表达式

[[逗号表达式]]中，逗号不是单纯的分隔符，而是一个有语义的运算符。

```cpp
int x = (1, 2, 3);
```

它的求值规则是：

- 先计算左边表达式。
- 再计算右边表达式。
- 整个逗号表达式的值是最右边表达式的值。

所以上例中，`x` 的值是 `3`。

在 `for` 循环里常用逗号表达式同时更新多个变量：

```cpp
for (int i = 0, j = 10; i < j; ++i, --j) {
    // ...
}
```

步进部分 `++i, --j` 会先执行 `++i`，再执行 `--j`。

> [!warning] 易错点
> 逗号表达式的结果是最后一个表达式的结果，不是所有表达式结果的集合。

## 赋值表达式与右结合

赋值表达式采用[[右结合]]。

```cpp
int x = 0;
int y = 0;

x = y = 100;
```

它等价于：

```cpp
x = (y = 100);
```

执行含义是：

1. 先计算右边的 `y = 100`，把 `100` 赋给 `y`。
2. `y = 100` 这个表达式产生结果。
3. 再把这个结果赋给 `x`。

因此最后 `x` 和 `y` 都是 `100`。

## 左值表达式

[[左值表达式]]可以粗略理解为：表达式结果对应一个可定位的存储位置，因此可以出现在赋值号左边。

例如：

```cpp
int x = 0;
x = 10; // x 是左值表达式
```

`x` 对应一块真实存在的存储空间，所以可以作为赋值目标。

再如数组元素：

```cpp
int a[3]{1, 2, 3};
a[0] = 10; // a[0] 是左值表达式
```

## 右值表达式

[[右值表达式]]通常只能用于求值，不能作为赋值目标。

例如：

```cpp
2 = 3; // 错误
```

字面值 `2` 只是一个值，不代表一个可以写入的存储位置，因此不能放在赋值号左边。

```cpp
int x = 0;
x + 1 = 3; // 错误
```

`x + 1` 计算出一个临时结果，也不能作为被赋值对象。

> [!important] 高频考点
> 判断一个表达式能否放在赋值号左边，关键看它是否代表一个可写的、真实存在的对象位置。

## Lambda 表达式

[[Lambda表达式]]是现代 C++ 引入的一种匿名函数表达式。

基本形式可以先记为：

```cpp
auto add = [](int a, int b) {
    return a + b;
};

int result = add(1, 2);
```

它的含义是：定义一个没有普通函数名的函数对象，并把它保存到变量 `add` 中。

课程此处只要求先认识：lambda 不只是 C++ 的概念，许多现代编程语言也支持类似的匿名函数写法。

## 本节速记

> [!summary] 速记
> 表达式通常有值，只是有时被丢弃。`for` 的三段都是表达式位置，`for (;;)` 是无限循环。逗号表达式从左到右求值，结果取最后一个；赋值表达式右结合，所以 `x = y = 100` 先做 `y = 100`。能放赋值号左边的通常是左值，只有临时结果或字面值通常是右值。
