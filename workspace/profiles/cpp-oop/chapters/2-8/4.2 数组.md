# 4.2 数组

## 本节核心

本节讲 C++ 中[[数组]]的基本形式、初始化方式、理解方法，以及[[指针数组]]和[[数组指针]]的区别。

- C++ 数组下标以 `0` 为起点。
- 数组大小在传统 C++ 数组声明中必须是[[编译期常量]]。
- 数组可以用等号初始化，也可以用[[花括号初始化]]。
- 理解数组声明时，可以把 `[]` 看成类型的一部分：`int[5] a` 表示 `a` 是 5 个 `int` 组成的数组。
- [[指针数组]]是“数组”，数组元素是指针。
- [[数组指针]]是“指针”，指向一个数组。

> [!important] 高频考点
> `int* p[5]` 和 `int (*p)[5]` 只差一对括号，但含义完全不同：前者是指针数组，后者是数组指针。

## 一维数组

一维数组的常见声明：

```cpp
int a[5];
```

含义是：`a` 是一个数组，有 5 个元素，每个元素都是 `int`。

数组下标从 `0` 开始：

```cpp
int a[5];

a[0] = 1; // 第 1 个元素
a[4] = 5; // 第 5 个元素
```

对于长度为 `5` 的数组，合法下标是 `0` 到 `4`。

> [!warning] 易错点
> `a[5]` 不是第 5 个元素，而是越过了最后一个合法元素。C++ 原生数组不会自动做安全边界检查。

## 数组初始化

数组可以使用初始化列表：

```cpp
int a[3] = {1, 2, 3};
```

C++11 以后也可以使用更统一的花括号写法：

```cpp
int a[3]{1, 2, 3};
```

对于简单整数数组，两种写法都容易理解。对于后续更复杂的对象数组，花括号初始化会更统一。

## 数组长度必须是编译期常量

传统 C++ 原生数组声明中，数组长度需要在编译期确定：

```cpp
const int n = 5;
int a[n]; // n 是编译期常量时可以
```

不能依赖运行时输入来声明普通栈上原生数组：

```cpp
int n;
cin >> n;
int a[n]; // 标准 C++ 中不应依赖这种写法
```

这里的关键是：编译器在编译数组声明语句时，需要知道数组大小。

## 如何理解数组类型

传统写法是：

```cpp
int a[5];
```

课程建议理解时可以把 `[]` 提到类型上：

```text
int[5] a
```

意思是：

- `a` 是一个变量名。
- `a` 的类型是“包含 5 个 `int` 的数组”。

这种理解方式在类型复杂时更清晰。写代码仍按 C++ 语法写 `int a[5];`，但读声明时可以把 `[]` 看成类型的一部分。

## 二维数组

二维数组常见写法：

```cpp
int matrix[2][3];
```

可以理解为：

```text
matrix 是一个数组；
它有 2 个元素；
每个元素又是一个包含 3 个 int 的数组。
```

初始化示例：

```cpp
int matrix[2][3]{
    {1, 2, 3},
    {4, 5, 6}
};
```

访问元素：

```cpp
matrix[0][0] = 10;
matrix[1][2] = 60;
```

## 指针数组

[[指针数组]]首先是数组。

```cpp
int* p[5];
```

读法是：`p` 是一个数组，有 5 个元素，每个元素都是 `int*` 指针。

可以按“把 `[]` 看成类型的一部分”来理解：

```text
int* [5] p
```

也就是：`p` 的类型是“包含 5 个 `int*` 元素的数组”。

## 数组指针

[[数组指针]]首先是指针。

```cpp
int (*p)[5];
```

读法是：`p` 是一个指针，指向一个包含 5 个 `int` 元素的数组。

括号很关键：

```cpp
int* p[5];    // 指针数组：p 是数组
int (*p)[5];  // 数组指针：p 是指针
```

示例：

```cpp
int a[5]{1, 2, 3, 4, 5};
int (*p)[5] = &a;

int value = (*p)[3];
```

`(*p)[3]` 的含义是：先通过 `*p` 得到 `p` 指向的整个数组，再访问下标为 `3` 的元素。

## 指针数组和数组指针对比

| 写法 | 名字本质 | 元素/指向对象 | 读法 |
|---|---|---|---|
| `int* p[5]` | `p` 是数组 | 每个元素是 `int*` | 指针数组 |
| `int (*p)[5]` | `p` 是指针 | 指向 5 个 `int` 组成的数组 | 数组指针 |

> [!tip] 识别方法
> 先找变量名，再看括号约束。`p[5]` 优先说明 `p` 是数组；`(*p)` 先说明 `p` 是指针。
