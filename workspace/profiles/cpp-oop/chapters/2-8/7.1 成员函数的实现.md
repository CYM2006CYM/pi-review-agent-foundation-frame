# 7.1 成员函数的实现

## 本节核心

[[类]] 中的 [[成员函数]] 表示对象的行为。类中通常先声明成员函数，然后在类外给出实现。

成员函数实现时，要解决两个问题：

- 这个函数属于哪个类？
- 函数内部访问的数据成员属于哪个对象？

第一个问题用 [[作用域解析符]] `::` 解决，第二个问题用隐含的 [[this指针]] 解决。

## 成员函数的种类

类中的成员函数有很多种，本节先从普通成员函数开始。

常见成员函数包括：

- [[普通成员函数]]
- [[常成员函数]]
- [[重载]] 的成员函数
- [[构造函数]]
- 析构函数
- 拷贝构造函数
- 赋值运算符函数
- 类型转换函数
- 静态成员函数

本节重点是普通成员函数的实现方式。

## 类定义与成员函数实现的位置

类定义通常放在 [[头文件]] 中：

```cpp
// Student.h
class Student {
public:
    Student();
    void study(int hours);
    void exercise();

private:
    int score_;
    int energy_;
};
```

成员函数的具体实现通常放在 `.cpp` 文件中：

```cpp
// Student.cpp
#include "Student.h"

void Student::study(int hours) {
    score_ += hours;
    energy_ -= 2;
}

void Student::exercise() {
    energy_ += 3;
}
```

这样可以把“接口声明”和“实现细节”分开。

## 作用域解析符 `::`

在类外实现成员函数时，需要写出：

```cpp
返回类型 类名::函数名(参数列表) {
    函数体
}
```

例如：

```cpp
void Student::study(int hours) {
    score_ += hours;
}
```

`Student::study` 表示：这里实现的是 `Student` 类中的 `study` 成员函数，而不是一个普通全局函数。

如果只写：

```cpp
void study(int hours) {
}
```

那它就是普通函数，不是 `Student` 的成员函数。

## 成员函数访问的是当前对象的数据

假设有两个学生对象：

```cpp
Student s1;
Student s2;
```

调用：

```cpp
s1.study(4);
s2.study(2);
```

同一个成员函数 `Student::study` 被不同对象调用时，修改的不是同一份数据。

- `s1.study(4)` 修改的是 `s1` 的 `score_`、`energy_`
- `s2.study(2)` 修改的是 `s2` 的 `score_`、`energy_`

那么函数体里的：

```cpp
score_ += hours;
```

到底是谁的 `score_`？

答案是：当前对象的 `score_`。

## this 指针是隐含参数

普通成员函数调用时，C++ 会隐含传入一个 [[this指针]]，它指向当前对象。

```cpp
s1.study(4);
```

可以理解为底层类似：

```cpp
Student::study(&s1, 4);
```

虽然源代码中只写了一个显式参数 `4`，但实际还隐含传入了当前对象地址。

因此：

```cpp
void Student::study(int hours) {
    score_ += hours;
}
```

可以理解为：

```cpp
void Student::study(int hours) {
    this->score_ += hours;
}
```

平时 `this->` 可以省略；省略后默认访问当前对象。

## 成员函数中调用其他成员函数

成员函数内部也可以直接调用同类的其他成员函数：

```cpp
void Student::study(int hours) {
    score_ += hours;
    energy_ -= 2;
    exercise();
}
```

这里的 `exercise()` 也默认是：

```cpp
this->exercise();
```

也就是说，它仍然作用于当前对象。

## 本节考点整理

- [[成员函数]] 表示对象行为，通常在类中声明、在 `.cpp` 中实现。
- 类外实现成员函数需要使用 [[作用域解析符]] `::`。
- `Student::study` 表示 `Student` 类中的 `study` 函数。
- 普通成员函数调用时有隐含的 [[this指针]]。
- `this` 指向当前调用该成员函数的对象。
- 成员函数中访问数据成员，默认访问当前对象的数据成员。
- `score_` 可理解为 `this->score_`。

## 本节速记

> 类外实现写 `类名::函数名`；  
> 成员函数自带 `this`；  
> 不写 `this->`，默认也是当前对象。
