# 10.2 二元运算符重载

## 本节核心

[[二元运算符重载]] 可以写成两种主要形式：

- [[非成员运算符重载]]：通常是自由函数或友元函数。
- [[成员运算符重载]]：运算符作为类的成员函数。

以向量加法为例：

```cpp
v1 + v2
```

它有两个操作数，所以是二元运算符。

## 自由函数形式

自由函数形式把左右两个操作数都写成显式参数：

```cpp
TVector operator+(const TVector& v1, const TVector& v2) {
    return TVector(v1.GetX() + v2.GetX(),
                   v1.GetY() + v2.GetY());
}
```

表达式：

```cpp
v1 + v2
```

会被理解成类似：

```cpp
operator+(v1, v2);
```

自由函数形式的特点：

- 左操作数和右操作数地位对称。
- 适合 `+`、`-`、`*`、`/`、`==`、`<` 等不一定要修改左侧对象的运算。
- 如果需要访问私有成员，可以声明为 [[友元函数]]。

## 成员函数形式

成员函数形式中，左操作数是当前对象，右操作数是显式参数：

```cpp
class TVector {
public:
    TVector operator+(const TVector& rhs) const {
        return TVector(x + rhs.x, y + rhs.y);
    }

private:
    int x;
    int y;
};
```

表达式：

```cpp
v1 + v2
```

会被理解成类似：

```cpp
v1.operator+(v2);
```

成员函数形式的特点：

- 左操作数必须能作为当前对象。
- 参数列表中只写右操作数。
- 如果运算需要修改当前对象，成员函数形式更自然。

## const 的位置

对于不修改对象的运算，例如普通加法：

```cpp
TVector operator+(const TVector& rhs) const;
```

两个 `const` 含义不同：

- 参数中的 `const TVector& rhs`：不修改右侧对象。
- 函数末尾的 `const`：不修改左侧当前对象。

普通 `+` 应该产生新结果，不应改变两个操作数，所以成员函数形式通常要加末尾 `const`。

## 返回类型

返回类型要符合内置类型语义。

普通加法：

```cpp
a + b
```

返回的是一个新值，不是 `a` 或 `b` 本身。因此对象的 `operator+` 通常按值返回：

```cpp
TVector operator+(const TVector& rhs) const;
```

赋值类运算，例如 `+=` 或 `=`，会修改左侧对象，通常返回左侧对象引用：

```cpp
TVector& operator+=(const TVector& rhs);
TVector& operator=(const TVector& rhs);
```

## 友元函数的使用

如果自由函数需要直接访问私有成员，可以在类中声明为友元：

```cpp
class A {
    friend A operator+(const A& lhs, const A& rhs);

private:
    int x;
    int y;
};

A operator+(const A& lhs, const A& rhs) {
    return A(lhs.x + rhs.x, lhs.y + rhs.y);
}
```

友元函数不是成员函数，但可以访问类的私有成员。

> [!warning] 易错点
> 友元声明不等于成员函数声明。友元函数没有隐含的 `this` 指针。

## 返回值优化

普通二元运算常常写成返回临时对象：

```cpp
return A(lhs.x + rhs.x, lhs.y + rhs.y);
```

从语义上看，这是按值返回。但现代编译器通常会做 [[返回值优化]]，减少临时对象带来的开销。

因此，普通 `+` 不必为了“省拷贝”而返回引用。返回局部临时对象的引用反而是严重错误。

## 本节考点整理

- 二元运算符有两个操作数。
- 自由函数形式：`operator+(lhs, rhs)`。
- 成员函数形式：`lhs.operator+(rhs)`。
- 成员函数中左操作数由 [[this指针]] 表示，参数只写右操作数。
- 不修改对象的成员运算符函数末尾应加 `const`。
- 普通 `+` 通常按值返回新对象。
- `+=`、`=` 这类修改左侧对象的运算通常返回左侧对象引用。
- 需要访问私有成员的自由函数可以声明为 [[友元函数]]。
