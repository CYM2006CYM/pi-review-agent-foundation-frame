# 第16章考点速记

## 一页总览

| 主题 | 必背结论 |
|---|---|
| [[向上类型转换]] | 派生类转基类，public 继承下天然安全 |
| [[向下类型转换]] | 基类转派生类，需要确认真实对象类型 |
| private/protected 继承 | 通常不适合做继承天然转换，因为不表达 is-a |
| public 继承 | 表示 is-a，派生类可当作基类使用 |
| 指针向上转换 | 安全，不复制对象，不丢派生类数据 |
| 引用向上转换 | 安全，不复制对象，不丢派生类数据 |
| 对象向上转换 | 安全，但发生[[对象切片]] |
| [[static_cast]] | 编译期静态转换 |
| [[const_cast]] | 添加或去除 `const`/`volatile` |
| [[reinterpret_cast]] | 底层重新解释，风险高 |
| [[dynamic_cast]] | 运行时类型检查，常用于安全向下转换 |

## 高频代码

向上转换：

```cpp
Derived d;
Base* pb = &d;     // 安全
Base& rb = d;      // 安全
Base b = d;        // 安全，但对象切片
```

对象切片：

```cpp
class Car {
public:
    void run();
};

class MTCar : public Car {
public:
    void shift();
};

MTCar mt;
Car c = mt;        // 只保留 Car 部分，shift 所在的派生类部分被裁剪
```

dynamic_cast 指针形式：

```cpp
Animal* animal = new Dog;
Dog* dog = dynamic_cast<Dog*>(animal);

if (dog != nullptr) {
    dog->run();
}
```

dynamic_cast 引用形式：

```cpp
Dog& dog = dynamic_cast<Dog&>(animalRef);  // 失败会抛异常
```

## 四个 cast 对比

| 操作符 | 关键词 | 失败或风险 |
|---|---|---|
| `static_cast<T>(x)` | 静态、编译期 | 不做运行时真实类型检查 |
| `const_cast<T>(x)` | 去 const、加 const、volatile | 修改真正的常对象会出问题 |
| `reinterpret_cast<T>(x)` | 重新解释 | 最危险，通常用于底层代码 |
| `dynamic_cast<T>(x)` | 动态、运行时、RTTI | 指针失败返回空，引用失败抛异常 |

## 高频辨析

| 易混点 | 正确理解 |
|---|---|
| public 继承 vs private 继承 | public 表示 is-a；private 常是实现复用 |
| 向上转换 vs 向下转换 | 向上从具体到一般；向下从一般到具体 |
| 指针/引用向上转换 vs 对象向上转换 | 前者不丢数据；后者会对象切片 |
| `static_cast` vs `dynamic_cast` | 前者编译期转换；后者运行时检查 |
| `const_cast` vs 修改常对象 | 去掉 const 不等于所有对象都能安全修改 |
| `reinterpret_cast` vs 普通转换 | 它是重新解释底层表示，不保证语义正确 |
