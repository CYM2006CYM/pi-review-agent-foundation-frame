# C++ 面向对象程序设计

这是随包发布的自包含 C++ 复习资料包，面向课程复习和班级共享使用。

本资料包不依赖仓库根目录的 `reference/`。安装后 pi-review 会把它从 `workspace/profiles/cpp-oop/` 种到用户可写的 profile 目录，同学可以直接运行 `/review` 选择本 profile 开始复习，不需要额外执行 `/review-init`。

## 复习目标

- 面向 C++ 面向对象程序设计课程复习。
- 以概念理解、代码判断、机制辨析、错因复盘为主。
- 支持概念卡片复习、直接做题、按章节学习三种模式。

## 资料结构

- `knowledge_index.json`：章节与知识点索引。
- `cards/`：概念卡片，用于模式 1。
- `chapters/`：章节笔记，用于模式 3。
- `exam_points/`：考点总结，用于模式 2。

## 出题风格

- 优先围绕知识点的常见误区出题。
- 选择题和判断题应给出明确答案，简答题应给出要点式解析。
- 代码相关题目尽量包含短小 C++ 片段。
