# 记忆与练习

## 核心概念

- **主动回忆**：在看答案前先尝试自己提取知识。
- **间隔复习**：把复习分散到多个时间点，降低遗忘。
- **交错练习**：混合不同类型的问题，训练识别和迁移。

## 知识点详解

主动回忆的关键不是“看懂”，而是在没有提示的情况下把知识取出来。重复阅读会带来熟悉感，但熟悉感不等于能独立使用。

间隔复习把复习安排在不同时间点，让大脑多次重新提取知识。一次长时间突击可能带来短期表现，但长期保持通常较弱。

交错练习把相近但不同的问题混在一起。它会让练习过程更难，但能训练学习者判断“这题属于哪一类”。

## 常见误区

- 把“看答案觉得懂了”当成掌握。
- 只在考试前集中复习。
- 同类题连续刷很多，却没有训练题型识别。
